HTML

JavaScript

React

A JavaScript runtime built on Chrome's V8 JavaScript engine, used for developing server-side and networking applications.
Express.js
A web application framework for Node.js, used for building web applications and APIs.
MongoDB
A source-available cross-platform document-oriented database program, used for high volume data storage.
