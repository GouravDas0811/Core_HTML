HTML tags

  1. href : Specifies the URL of the page the link goes to.
  2. src Specifies the path to the image to be displayed.
  3. alt : Specifies an alternate text for an image, if the image for some reason cannot be displayed.
  4. style : Used to add styles to an element, such as color, font, size, and more.
  5. lang : Should always be included inside the tag, to declare the language of the Web page.
